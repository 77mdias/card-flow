# Development Tasks

Este diretorio concentra specs e documentacao de execucao das tasks em desenvolvimento.

Arquivos atuais:
- `task-s01-001-auth0-base-user-lifecycle.md`: spec oficial da task.
- `task-s01-001-auth0-base-user-lifecycle-features.md`: documentacao funcional/tecnica da entrega.

Padrao recomendado para novas tasks:
1. Criar a spec com base em `docs/tasks/templates/task-spec.md`.
2. Atualizar status e criterios de aceite apos implementacao.
3. Gerar um documento de features entregues com:
   - fluxo funcional;
   - contrato tecnico;
   - passos de validacao;
   - limites da entrega.
