# Templates

Este diretorio contem modelos padrao para operacao do projeto.

Arquivos:
- `task-spec.md`: especificacao completa de task.
- `bug-report.md`: modelo padrao para reporte de erro.

Regra:
- Sempre copiar o template para o destino da task/issue.
- Nao editar o template original sem alinhamento de processo.
