# Sprint 04 - Dashboard Multi-Moeda

## Objetivo
Entregar visao consolidada de gastos por moeda no MVP.

## Escopo
- totais por moeda.
- lista de faturas abertas.
- ultimas 10 transacoes.
- acao de marcar statement como paga.

## Criterios de aceite
- dashboard nao mistura moedas no mesmo total.
- dados privados sem cache compartilhado.
- acao de pagamento atualiza status da fatura.

## Fora de escopo
- conversao automatica de moeda.
